* Package Management

This section configures Emacs’ built-in package system and defines the package archives used to install and manage extensions. It bootstraps use-package, which provides a consistent and declarative way to install, configure, and organize packages. Enabling org-tempo restores Org structure templates, improving the authoring experience in Org-mode–based configurations.


#+begin_src emacs-lisp
;; ------------------------------
;; Package manager setup
;; ------------------------------

(require 'package)

(setq package-archives
      '(("melpa" . "https://melpa.org/packages/")
        ("org"   . "https://orgmode.org/elpa/")
        ("elpa"  . "https://elpa.gnu.org/packages/")))

(unless package--initialized
  (package-initialize))

(unless package-archive-contents
  (package-refresh-contents))

#+end_src

* Bootstrap

This bootstrap section ensures that the Emacs package system is initialized correctly on first startup. It configures the required package archives, refreshes package metadata when needed, and installs use-package if it is not already present. By handling these prerequisites upfront, the rest of the configuration can rely on use-package declarations without manual setup or ordering concerns.

#+begin_src emacs-lisp
;; ------------------------------
;; use-package bootstrap
;; ------------------------------

(unless (package-installed-p 'use-package)
  (package-install 'use-package))

(require 'use-package)
(setq use-package-always-ensure t)

;; Enable Org structure templates (e.g. <s TAB)
(require 'org-tempo)
#+end_src

* Automatic Package Updates
This section enables automated package maintenance using auto-package-update. Packages are periodically checked and updated on a weekly schedule, with optional user confirmation before updates are applied. This helps keep the Emacs environment secure and up to date while minimizing manual intervention and startup noise.

#+begin_src emacs-lisp
;; ------------------------------
;; Auto package updates
;; ------------------------------

(use-package auto-package-update
  :custom
  (auto-package-update-interval 7)
  (auto-package-update-prompt-before-update t)
  (auto-package-update-hide-results t)
  :config
  (auto-package-update-maybe)
  (auto-package-update-at-time "09:00"))
#+end_src
